## What is MasterDataCleaner?

**MasterDataCleaner** is an intelligent, step-by-step data cleaning solution designed for professional environments. It empowers users to transform messy, inconsistent data into structured, high-quality information through an AI-guided assistant.

### Key Capabilities

- **AI-Powered Analysis** - Automatically detect anomalies, duplicates, and formatting errors
- **Step-by-Step Workflow** - Logical, linear process ensuring no data validation step is missed
- **Multiple Data Sources** - Support for files (CSV, Excel, JSON), databases, and clipboard
- **Customizable Logic** - Powerful scripting capabilities for complex transformations
- **Real-Time Preview** - See changes before committing them
- **Undo/Redo** - Full operation history with rollback capability
