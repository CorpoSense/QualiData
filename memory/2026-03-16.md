## Pagination Work (Ongoing)

### Key Findings

**BTable Pagination in bootstrap-vue-next (v0.43.7):**
- BTable does NOT have a `total-rows` prop
- Pagination works via `:per-page` and `:current-page` props
- For LOCAL pagination: BTable slices the items array automatically
- For SERVER pagination: Need to use a provider function OR fetch new data on page change

**Current Issue:**
- Dataset preview is cached (500 rows max)
- BTable sees only the items in the `data` array, not total rows
- Without provider function, BTable can only paginate within the loaded data

### Solutions to Try
1. Use BTable with a provider function that fetches data on page change
2. Or: Keep custom pagination buttons but fetch new data from API on page change
