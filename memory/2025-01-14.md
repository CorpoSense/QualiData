# Serper Search Plugin Development

## Date
2025-01-14

## Summary
Created a Serper search plugin for OpenClaw to provide `serper_search` tool using Serper Google Search API.

## Key Files
- Plugin location: `~/.openclaw/extensions/serper/`
- Files: `openclaw.plugin.json`, `index.ts`, `package.json`
- Tool name: `serper_search`

## Configuration
- SERPER_API_KEY set in env (length 40)
- Also stored in `/root/.openclaw/openclaw.json` config
- Plugin enabled via `plugins.entries.serper.enabled: true`

## Issues Resolved
- **TypeBox issue**: Plugin initially used `@sinclair/typebox` which caused "Cannot find module" error at runtime
- **Fix**: Changed to plain JSON schema for tool parameters instead of TypeBox

## Status
- Gateway restarted successfully
- Plugin now loads correctly ✅
- serper_search tool is working ✅ (tested with "OpenClaw AI assistant" query)
