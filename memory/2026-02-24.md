## MasterDataCleaner SaaS Specification Session

User provided detailed specs for MasterDataCleaner SaaS:

### Core Architecture Decisions
- Data: Memory (pandas) during work → Database (PostgreSQL/MySQL) for persistence
- Storage limits by tier: Free (1K rows, 5MB), Pro (50K rows, 100MB), Enterprise (500K rows, 1GB)
- Import/Export: CSV, Excel, JSON + SQLAlchemy database connections
- Agents: Reusable across projects, pre-built templates included

### Key Features Spec'd
1. **Standard Operations:** Add/Remove/Rename/Merge/Split columns
2. **Simple Cleaning:** Strip, case transforms, duplicates, missing values, format standardization
3. **AI Cleaning:** Agent-based with batch processing (default 10 rows), structured JSON output
4. **Custom Functions:** Predefined library only (no arbitrary Python for security)
5. **Preview:** Before/After with 25/50/100 row options
6. **History:** 10 operation undo/redo stack

### Tech Stack Finalized
- Frontend: Vue 3 + Pinia + Buefy + Vite
- Backend: FastAPI + SQLAlchemy + pandas + LangChain
- Database: PostgreSQL (Supabase/NeonDB) or MySQL (Aiven)
- Billing: Lago
- AI: OpenAI, Anthropic, Google, Ollama, Groq, DeepSeek, OpenRouter

### Pricing Tiers
- **Free:** 1 project, 1K rows, all features
- **Pro ($19/mo):** 10 projects, 50K rows
- **Enterprise ($99/mo):** Unlimited projects, 500K rows, collaboration
